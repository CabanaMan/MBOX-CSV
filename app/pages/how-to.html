Email export guide

Follow these instructions to download an MBOX or PST archive from your email provider. Once you have the file, upload it to mbox-csv.com and we will convert the messages to a spreadsheet.

Before you export

Most providers let you download your email as either .mbox or .pst files. Both formats are supported. When possible, choose the option to split the export into multiple files so each archive is under 20 GB.

If your provider only offers PST (Microsoft Outlook), you can still upload it directly—our backend converts PST to MBOX automatically before generating the CSV.

Step-by-step export instructions

Gmail / Google Workspace

  1. Open Google Takeout and click Deselect all.
  2. Enable Mail, then choose All Mail data included to filter labels if needed.
  3. Pick .zip as the delivery format, choose a large file size split (10 GB or higher), and start the export.
  4. When the archive is ready, download and unzip it. The .mbox files are inside the Mail folder.

Outlook (Microsoft 365 / Outlook.com)

  1. Launch Outlook on Windows or macOS.
  2. Go to File → Open & Export → Import/Export.
  3. Select Export to a file → Outlook Data File (.pst) and choose the folders to include.
  4. Save the export. Upload the resulting .pst file here for conversion.

Outlook.com privacy dashboard

  1. Visit the Microsoft privacy dashboard and sign in.
  2. Open Download your data, create a new export, and enable Mail.
  3. Microsoft emails you when the export is ready. Download the archive and extract the .pst file.

Apple Mail (macOS)

  1. In Mail, select the mailbox or folder you want to export.
  2. Choose Mailbox → Export Mailbox…, pick a destination folder, and click Choose.
  3. macOS saves a ready-to-upload .mbox directory containing your messages.

Mozilla Thunderbird

  1. Install the free ImportExportTools NG add-on from the Thunderbird extensions catalog.
  2. Right-click the folder you want to export and choose ImportExportTools NG → Export folder.
  3. Select the MBOX format and save the file.

Proton Mail

  1. Download the Proton Import-Export app (available on paid plans) from the settings page.
  2. Sign in, pick the mailboxes to archive, and choose MBOX as the export format.
  3. The tool creates encrypted archives; decrypt them with your password and upload the resulting .mbox files.

Yahoo Mail

  1. Open the Yahoo Privacy Dashboard.
  2. Create a new request under Download & view your data and choose Mail.
  3. When Yahoo emails the download link, extract the archive to find the .mbox file.

Zoho Mail

  1. In Zoho Mail, go to Settings → Import/Export.
  2. Select Export, pick the folder you need, and choose the MBOX format.
  3. Submit the export request and download the archive when it is ready.

iCloud Mail

  1. Add your iCloud account to Apple Mail on macOS if it is not already present.
  2. Select the mailbox, then click Mailbox → Export Mailbox….
  3. Save the folder—inside you will find a standard .mbox archive.

After the export

Once you have your archive:

  1. Upload the file on the homepage. Jobs up to 20 GB are supported.
  2. Leave the tab open while we parse the messages. Progress updates appear automatically.
  3. When the conversion is complete, download the emails.zip package containing emails.csv.
