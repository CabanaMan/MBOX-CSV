Export email to MBOX or PST

Export your mailbox using the steps below, then convert to CSV.

Gmail → MBOX

  1. Open Google Takeout → deselect all → enable Mail.
  2. Optional: “All Mail data included” to pick labels.
  3. Choose .zip and a large file size split.
  4. Export, download, and extract the archive to get .mbox files.

Microsoft Outlook (Microsoft 365 / Outlook.com / Exchange) → PST

  1. Use Outlook desktop (Windows/Mac).
  2. File → Open & Export → Import/Export → Export to a file → Outlook Data File (.pst).
  3. Select mailbox/folders → choose a save location → finish.
  4. Upload the .pst here for conversion.

Apple Mail (macOS) → MBOX

  1. In Mail, select a mailbox/folder.
  2. Mailbox → Export Mailbox… → choose a folder → you get a .mbox.

Thunderbird → MBOX

  1. Install “ImportExportTools NG”.
  2. Right-click a folder → Export folder → choose MBOX.

Proton Mail → MBOX

  1. Use Proton’s Import-Export app (paid plans).
  2. Export selected mailboxes to get .mbox.

Yahoo Mail → MBOX

  1. Use Yahoo Data Download or add the account to Thunderbird and export as MBOX.

Zoho Mail → MBOX

  1. Settings → Import/Export → Export → pick folder → format MBOX.

iCloud Mail → MBOX

  1. Add iCloud to Apple Mail (macOS) and use Mailbox → Export Mailbox….
