How to get your email archive

This guide explains how to export an archive that you can upload to mbox-csv.com.

Gmail

  1. Open Google Takeout.
  2. Deselect all, select Mail.
  3. Create export and download the .mbox.

Microsoft Outlook / Office 365

  1. File → Open & Export → Import/Export → Export to a file → Outlook Data File (.pst).
  2. Upload .pst here; we convert using readpst.

Yahoo Mail

  1. Privacy Dashboard → Download My Data → Mail.

Apple Mail (macOS)

  1. Mailbox → Export Mailbox… → gets .mbox.

We support .mbox directly and .pst via server-side conversion.
