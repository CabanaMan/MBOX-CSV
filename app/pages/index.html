How to export an MBOX from popular providers

Use these quick steps to grab the right archive from your mailbox. Need more detail? Read the full email export guide.

Gmail

Google Takeout lets you export every label as an MBOX.

  1. Visit takeout.google.com and deselect all services.
  2. Enable Mail → “All Mail data included” to pick labels if needed.
  3. Choose a .zip export, start the archive, then download and unzip the resulting .mbox files.

Outlook.com / Hotmail

Request a complete mailbox export from Microsoft’s privacy dashboard.

  1. Open account.microsoft.com/privacy → Download your data.
  2. Create a new export and choose Mail → Microsoft emails you when the PST is ready.
  3. Download the archive and upload the included .pst or .mbox file for conversion.

Microsoft 365 / Outlook desktop

Use the classic Import/Export wizard.

  1. In Outlook, go to File → Open & Export → Import/Export.
  2. Select Export to a file → Outlook Data File (.pst).
  3. Choose the mailbox or folder, pick a destination, and upload the resulting .pst.

Yahoo Mail

Yahoo offers a data download tool for full mailbox exports.

  1. Go to the Yahoo Privacy Dashboard → Download & view your data.
  2. Request a Mail export and wait for the email confirmation.
  3. Download the archive and extract the .mbox inside.

Apple Mail & iCloud

Export directly from the Mail app on macOS.

  1. Select the mailbox or folder in Mail.
  2. Choose Mailbox → Export Mailbox… and save the folder.
  3. The export folder contains a ready-to-upload .mbox file.

Thunderbird

Install a free add-on to save folders as MBOX.

  1. Install the ImportExportTools NG extension.
  2. Right-click a folder → Export folder → choose MBOX.
  3. Repeat for additional folders or use “Export all folders” to batch them.

Proton Mail

Paid plans unlock their Import-Export application.

  1. Download the Proton Import-Export tool from your account settings.
  2. Sign in, select the mailboxes to export, and choose the MBOX format.
  3. Upload the generated archive here once the export completes.

Zoho Mail

Zoho’s admin console can export any mailbox.

  1. Open Settings → Import/Export.
  2. Select Export, pick the folder or account, and choose MBOX.
  3. Start the export and download the .mbox file when notified.

Frequently asked questions

Can I convert MBOX files created by Google Takeout?

Yes. Google Takeout exports Gmail mailboxes as standard MBOX archives that work perfectly with this converter.

Is there a limit to how many times I can use the tool?

You can submit multiple jobs. Each upload can be up to 20 GB, and you can start another conversion as soon as the previous one finishes.

What if I need PST or EML support?

The CSV output focuses on core headers today. Contact us if you need PST ingestion or body text exports — we are iterating quickly based on feedback.
